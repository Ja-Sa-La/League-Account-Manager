# Change Name

## Overview

The change name window collects input to update a Riot ID.

## Functions

- Prompt for a new name and tagline.
- Validate availability and submit the change.

## Options

- `Name`: Riot ID name field.
- `Tagline`: Riot ID tagline field.
- `Check if available`: validate the Riot ID.
- `Change Name`: submit the change.
- Link: `Check available characters for RiotID`.

## Tutorial

1. Open the change name window from the relevant action.
2. Enter the new name and tagline.
3. Use `Check if available`, then `Change Name` to apply.

## Technical details

- Window: `Windows/ChangeName.xaml`
- Code-behind: `Windows/ChangeName.xaml.cs`
- Uses dialog-style controls and validation logic.
