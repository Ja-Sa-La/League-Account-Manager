# Progress Window

## Overview

The progress window displays progress for long-running tasks.

## Functions

- Show task status text and a progress bar.

## Options

- Status text: describes the current operation.
- Progress bar: visual indicator of progress.

## Tutorial

1. Start a long-running action that opens the progress window.
2. Monitor the status and progress while the task runs.
3. Close the window when the task completes.

## Technical details

- Window: `Windows/ProgressWindow.xaml`
- Code-behind: `Windows/ProgressWindow.xaml.cs`
- Uses WPF progress indicators and status text.
