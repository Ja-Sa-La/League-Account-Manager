# Password Prompt

## Overview

The password prompt window collects a password for a protected action.

## Functions

- Request a password input.
- Confirm or cancel the prompt.

## Options

- Password box: enter the account file password.
- `OK`: confirm and continue.
- `Cancel`: close without proceeding.

## Tutorial

1. Enter the password when the prompt appears.
2. Select `OK` to proceed or `Cancel` to abort.

## Technical details

- Window: `Windows/PasswordPrompt.xaml`
- Code-behind: `Windows/PasswordPrompt.xaml.cs`
- Uses a password box and dialog result handlers.
