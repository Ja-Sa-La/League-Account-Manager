# Missing Info

## Overview

The missing info window requests required information before continuing.

## Functions

- Prompt for missing fields.
- Confirm the input to proceed.

## Options

- `username`: required username field.
- `pass`: required password field.
- `OK`: confirm and continue.

## Tutorial

1. When prompted, fill in the missing fields.
2. Select `OK` to continue.

## Technical details

- Window: `Windows/MissingInfo.xaml`
- Code-behind: `Windows/MissingInfo.xaml.cs`
- Uses form fields with basic validation logic.
