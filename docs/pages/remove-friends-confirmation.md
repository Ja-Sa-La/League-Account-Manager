# Remove Friends Confirmation

## Overview

The remove friends confirmation window confirms a removal action.

## Functions

- Display a confirmation message.
- Accept or cancel the action.

## Options

- `Remove accounts`: confirm bulk friend removal.
- `Cancel`: close the dialog without removing friends.

## Tutorial

1. Review the confirmation message.
2. Use `Remove accounts` to proceed or `Cancel` to abort.

## Technical details

- Window: `Windows/RemoveFriendsConfirmation.xaml`
- Code-behind: `Windows/RemoveFriendsConfirmation.xaml.cs`
- Uses dialog buttons wired to confirmation logic.
