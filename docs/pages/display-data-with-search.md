# Display Data With Search

## Overview

This window displays a data set with a search box for filtering.

## Functions

- Show data in a read-only text area.
- Filter results using a search input.

## Options

- Search box: filter the displayed data.
- Data viewer: read-only text area with selectable content.

## Tutorial

1. Open the window from the action that launches it.
2. Type in the search box to filter results.
3. Review the filtered data in the viewer.

## Technical details

- Window: `Windows/DisplayDataWithSearch.xaml`
- Code-behind: `Windows/DisplayDataWithSearch.xaml.cs`
- Uses WPF list and search inputs to filter items.
