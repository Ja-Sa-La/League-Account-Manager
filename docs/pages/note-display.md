# Note Display

## Overview

The note display window shows and edits notes.

## Functions

- Render a note body.
- Allow editing of note content.

## Options

- Notes text area: editable multi-line content.

## Tutorial

1. Read or update the note displayed in the window.
2. Close the window to return to the prior page.

## Technical details

- Window: `Windows/NoteDisplay.xaml`
- Code-behind: `Windows/NoteDisplay.xaml.cs`
- Uses a simple content layout and a close handler.
